# NBA Era Analysis & MVP Prediction Model

In this project, I acted as a **Data Analyst** for an NBA franchise, tasked with cleaning a messy 27-year dataset and extracting actionable insights.

### **Key Highlights**
* **Advanced Data Cleaning:** Successfully cleaned over **12,000 records** and fixed critical data quality issues regarding "Undrafted" players.
* **Feature Engineering:** Engineered a new **"MVP Score"** feature to better evaluate player value.
* **Technical Implementation:** Utilized **Advanced SQL** techniques, including:
    * Window Functions
    * CTEs (Common Table Expressions)
    * Self-Joins
* **Final Deliverables:** Produced a data-driven "Dream Team" roster and a comparative analysis of physical evolution in the league from the 1990s to the 2020s.
